Sobre o Projeto

Sistema de Aprendizagem da Linguagem GIT e Plataforma Git Hub.

Os estudantes de cursos de informática encontram dificuldades de aprendizagem e manuseio de linguagens como Git e plataforma Git hub que auxiliam em seu trabalho. Dúvidas de linhas de comandos, execução de versionamentos e mesmo compartilhamento de projetos para outras pessoas.
Com isso, o projeto consiste no desenvolvimento de um site que possibilite no desenvolvimento desses novos profissionais capacitando-os para o mercado de trabalho de forma fácil e intuitiva.
A elaboração do projeto dispõe de recursos financeiros limitados e a colaboração de patrocinadores para que todas as funções sejam implementadas.

Para este projeto, fica nomeado Roberto Hiroshi Watanabe como gerente, tendo ele as responsabilidades: fazer o planejamento, detalhando o escopo do projeto, definir o corpo da equipe que trabalhará neste projeto, manter a coordenação da equipe envolvida no projeto e fazer o acompanhamento para que nada fuja do escopo, manter o orçamento e o tempo do projeto que foi especificado no início do contrato. Deverá ser feita a alocação correta dos recursos disponíveis, sejam eles de custo, tempo ou pessoal, para que o resultado seja o mais esperado.
Em caso de atraso ou qualquer alteração, o gerente deverá emitir um relatório para o setor de controle e mudança para que possa ser alocado mais recursos para a finalização do projeto.

Para a realização do projeto, é necessário um grupo de especialistas em áreas como: professor na área de tecnologia da informação para o auxílio ao conteúdo que será colocado no site. Para elaboração do site será feita uma pesquisa para saber quais as principais dúvidas dos alunos, para uma melhor elaboração do conteúdo proposto no site.

Sistema para auxiliar na aprendizagem de linguagens de programação como Git e plataforma Git Hub. Auxiliando o aprendizado de estudantes e profissionais que estão se especializando para uma melhor vaga de trabalho.
